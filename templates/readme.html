NLP Virtual Lab

Overview

The NLP Virtual Lab is an interactive web-based educational platform designed to teach Natural Language Processing concepts through hands-on experiments and visualizations. Built with Flask and SQLAlchemy, it provides a comprehensive learning environment with modules covering various NLP techniques, real-time text processing, interactive quizzes, and visual demonstrations.

User Preferences

Preferred communication style: Simple, everyday language.

System Architecture

Backend Architecture

  • Framework: Flask web framework with Python
  • Database: SQLAlchemy ORM with SQLite (configurable)
  • Session Management: Flask sessions with secret key
  • Proxy Support: ProxyFix middleware

Frontend Architecture

  • Template Engine: Jinja2 with Bootstrap 5
  • CSS Framework: Bootstrap + custom dark theme
  • JavaScript Libraries: Feather Icons, Chart.js, D3.js
  • Interactive Components: AJAX-based real-time text processing

NLP Processing Engine

  • Core Library: NLTK
  • Capabilities: Tokenization, POS tagging, sentiment analysis, NER, stemming, lemmatization
  • Auto-setup: Downloads NLTK data packages

Key Components

Database Models

  • User Model: Authentication + Flask-Login
  • UserProgress: Tracks module scores and completion
  • QuizQuestion: Multiple-choice quizzes with explanations

Lab Modules

  • Text Preprocessing
  • Part-of-Speech Tagging
  • N-Gram Modeling
  • Named Entity Recognition
  • Sentiment Analysis
  • Text Classification
  • Word Embeddings
  • Chunking & Parsing

Interactive Features

  • Real-time text processing
  • Progress tracking
  • Quizzes per module
  • Charts and syntax tree visualizations

Deployment Strategy

Development

  • Flask dev server with debug mode
  • SQLite used locally
  • ENV variables: DATABASE_URL, SESSION_SECRET

Production

  • WSGI Server + ProxyFix
  • PostgreSQL via config
  • SQLAlchemy connection pooling
  • Session security with secret keys

Environment Variables

  • DATABASE_URL: DB connection
  • SESSION_SECRET: Session key
  • Host/Port configurable

The architecture emphasizes modularity, scalability, and simplicity for easy learning and deployment.
